# midi-toolbox
toolbox for different functions on specified midi sequeneces, as the first 'Grundlagen des Programmierens 1' project exercise

## functions
- [x] midi to frequency / frequency to midi
- [ ] midi to pitch class
- [ ] get octave
- [ ] generate scale
- [ ] generate chords
- [ ] transpose
- [ ] arpeggiate
- [ ] invert
- [ ] scale timing
- [ ] count midi notes
- [ ] get subsequent interval
- [ ] count intervals
